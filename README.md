# Портфолио системного аналитика

Специализируюсь на проектировании архитектуры backend-систем, интеграции сервисов (REST API, брокеры сообщений) и создании прозрачной технической документации.

###  Мой стек технологий
- **Анализ и Моделирование:** UML (Sequence, Use Case, Activity), BPMN 2.0, ER-диаграммы.
- **Интеграции и API:** REST (Swagger/OpenAPI), JSON, XML, понимание работы очередей (Kafka, RabbitMQ).
- **Базы данных:** PostgreSQL (проектирование схем, оптимизация сложных SQL-запросов, оконные функции, индексы).
- **Инструменты:** Jira, Confluence, Git, Miro, PlantUML.
