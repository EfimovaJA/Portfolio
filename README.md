# Портфолио системного аналитика

Специализируюсь на проектировании архитектуры backend-систем, интеграции сервисов (REST API, брокеры сообщений) и создании прозрачной технической документации.

###  Мой стек технологий
- **Анализ и Моделирование:** UML (Sequence, Use Case, Activity), BPMN 2.0, ER-диаграммы.
- **Интеграции и API:** REST (Swagger/OpenAPI), JSON, XML, понимание работы очередей (Kafka, RabbitMQ).
- **Базы данных:** PostgreSQL (проектирование схем, оптимизация сложных SQL-запросов, оконные функции, индексы).
- **Инструменты:** Jira, Confluence, Git, Miro, PlantUML.

### Примеры моих работ

1. **[Loyalty App Integration](ссылка_на_репозиторий)** — Проектирование интеграции программы лояльности для мобильного приложения. Полная трассировка требований, UML Sequence-диаграммы, спецификации JSON-контрактов и edge-кейсы возвратов.
2. **[TravelTech Fallback Architecture](ссылка_на_репозиторий)** — Техническое решение и ТЗ для отказоустойчивого рекомендательного шлюза (Паттерн Circuit Breaker, Fallback-сценарии с Rule-Based ранжированием при деградации ML-моделей).
3. **[SQL & Data Vault Optimization](ссылка_на_репозиторий)** — Песочница с примерами сложных аналитических SQL-запросов и спроектированной схемой данных по методологии Data Vault 2.0.
